# Theme Switcher (Python Tkinter)

This is a basic practice mini project built using Python and Tkinter.  
It provides a simple interface to switch between Light, Dark, and Auto themes.  
The project is made for learning and improving Python GUI development skills.

## Features
- Light mode
- Dark mode
- Auto mode (changes based on system time)
- Preview area to show the active theme
- Status message for each mode
